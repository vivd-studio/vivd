Plugins let a Vivd project do more than render pages. In the current product surface, the main plugin workflows are Contact Form and Analytics.

## When to configure plugins

- Set them up before the final publish pass.
- Revisit them whenever you change how leads, traffic tracking, or project ownership should work.

## Available guides

- [Contact Form](/plugins/contact-form/) explains activation, recipient setup, and embedding expectations.
- [Analytics](/plugins/analytics/) explains how to enable tracking and interpret the dashboard.

## Recommended workflow

1. Finish the main site structure in Studio.
2. Configure the plugins that matter for launch.
3. Test the form flow and analytics expectations.
4. Run through the publish checklist.
