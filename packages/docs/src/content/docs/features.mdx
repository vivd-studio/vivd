Vivd is built to cover the whole website workflow instead of only the first prompt.
You can start a project, refine it in a real workspace, add launch-critical platform
capabilities, and publish the result without stitching together a separate stack of
tools around it.

## Core product capabilities

- Start from scratch with a business brief, design references, and brand assets.
- Import an existing website or ZIP when you already have source material to adapt.
- Edit inside Studio with AI chat, direct text editing, live preview, and the asset/file explorer.
- Create new images with AI from the asset workflow.
- Edit and replace project imagery with AI-assisted image workflows.
- Configure first-party plugins such as Contact Form and Analytics inside the same project flow.
- Publish the draft to the live domain and keep self-hosting available when you need it.

## AI image creation and editing

Vivd treats image work as part of the same project workflow instead of a separate tool
handoff.

- Create new imagery from prompts and project references.
- Reuse project assets as visual references when generating replacements.
- Review and replace project images from the asset workflow without leaving Studio.
