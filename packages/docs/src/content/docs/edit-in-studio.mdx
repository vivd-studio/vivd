## What Studio is best at

- Large structural changes through the AI chat.
- Element-level refinement when you want to target a specific section.
- Direct text edits for fast corrections.
- Asset replacement and organization without leaving the project.

## Typical editing loop

1. Open the project in Studio.
2. Review the page visually from top to bottom.
3. Use the chat for larger layout, copy, or style changes.
4. Use direct text editing for precise wording fixes.
5. Replace or upload assets where the draft still feels generic.
6. Preview again and repeat until the project is ready to publish.

## Draft versus live

- **Draft / Studio** is your working version.
- **Live** is what visitors see on the published domain.

Changes in Studio do not affect the live site until you publish again.

## Good habits

- Make one focused pass for layout and one for copy.
- Replace weak placeholder images before polishing smaller details.
- Re-check forms, calls to action, and legal/contact information before publishing.
