# Commercial Licensing

This file is a plain-language summary of Vivd's current licensing model. The
binding terms are in [LICENSE](LICENSE).

Vivd is source-available under BUSL-1.1 with a Vivd-specific Additional Use
Grant. It is not licensed under an OSI-approved open-source license.

## Free use under the Additional Use Grant

You can use Vivd without a separate commercial license when all of the
following are true:

- The deployment is single-tenant and dedicated to one business, organization,
  or corporate group under common control, or to one client of a service
  provider.
- The deployment is not shared across unrelated customers.
- The deployment matches Vivd's `solo` install profile, or is substantially
  similar in practice.

This includes commercial single-tenant self-hosting, agencies or freelancers
running a dedicated instance for one client, consulting, support, maintenance,
and paid implementation work around Vivd.

Generated website output created with Vivd is not itself the Licensed Work and
is not restricted by the Vivd software license.

## Uses that require a separate commercial license

The following require a separate commercial agreement with the Licensor:

- Any multi-tenant, multi-organization, or shared-control-plane deployment.
- Any use of Vivd's `platform` install profile, or a substantially similar
  host-based SaaS configuration.
- Any hosted or managed Vivd service provided to multiple customers or
  unrelated end users from a shared deployment.
- Any white-label, OEM, embedded, or other productized commercial distribution
  of Vivd where the value derives substantially from Vivd functionality for
  multiple unrelated customers.
